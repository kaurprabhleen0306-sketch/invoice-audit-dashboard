# 📊 Invoice Audit Intelligence Dashboard

## 🚀 Live Demo
🔗 https://curious-puppy-fcd661.netlify.app/

## 📌 Project Overview
This project is an **Invoice Audit Intelligence Dashboard** designed to identify overbilling and financial discrepancies in invoice data.

The dashboard compares:
- **Billed Rate**
- **Agreed Contract Rate**
- **Quantity**

and calculates potential **overcharge** using the formula:

**Overcharge = (Billed Rate - Agreed Rate) × Quantity**
